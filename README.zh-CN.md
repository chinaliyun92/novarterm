# NovarTerm

[English](README.md) | [简体中文](README.zh-CN.md)

面向开发者与 DevOps 团队的 AI 优先终端工作空间。

NovarTerm 是一款 AI 原生桌面终端，把 AI 助手放在你执行命令、查看日志、传输文件、管理远程会话的同一个工作界面里。

![](docs/screenshots/06.png)

## 路线图与状态

NovarTerm 当前详细能力路线图如下：

| 类目 | 说明 | 状态 |
|---|---|---|
| AI | 支持使用 `Command + K` 快速打开/关闭 AI Command Bar，并可在同一工作区持续对话。 | ✅ |
| AI | 在 Shell 中右键 `Explain with AI` 可自动携带终端上下文发起解释。 | ✅ |
| AI | 终端异常输出支持智能识别，并提供 `Explain Error` 与 `Debug With AI` 快捷入口。 | ✅ |
| AI | 统一工作流内支持 `Explain`、`Run`、`Insert` 三种动作模式。 | ✅ |
| AI | 内置基础 AI 模型，支持用户添加自定义模型。 | ✅ |
| AI | AI Command Bar 支持快速切换自定义模型。 | ✅ |
| AI | 支持在终端内快速辅助生成命令（内容以 `>` 开头）。 | ✅ |
| Workspace | 多分屏支持水平分屏、垂直分屏与关闭当前分屏。 | ✅ |
| Workspace | 支持拖拽 pane 标题栏调整分屏布局，并在上/下/左/右位置提供放置提示。 | ✅ |
| Workspace | 支持应用重启后恢复 Tab/Panes 工作区布局。 | ✅ |
| Workspace | 新建 pane 操作支持继承上下文目录。 | ✅ |
| SSH | 支持在设置中保存多个服务器连接信息。 | ✅ |
| SSH | 支持在终端右键选择在当前 pane、新 pane 或新 tab 中快速连接。 | ✅ |
| SSH | 连接失败与状态变化提供可操作反馈，便于重试或切换。 | ✅ |
| 文件面板 | 支持在终端右键菜单中快速打开本地或远程服务器文件浏览面板。 | ✅ |
| 文件面板 | 支持在文件浏览面板内快速切换服务器/数据源。 | ✅ |
| 文件面板 | 支持在远程服务器文件面板中快速上传文件与目录。 | ✅ |
| 文件面板 | 支持在文件浏览面板中快速下载远程文件。 | ✅ |
| 文件面板 | 支持在文件浏览面板中快速新建文件、新建目录、删除文件、解压 ZIP 文件。 | ✅ |
| 拖拽 | 本地终端支持快速解析并插入拖入文件/目录的路径。 | ✅ |
| 拖拽 | 远程文件面板支持通过拖拽上传文件与目录。 | ✅ |
| 传输中心 | 支持统一查看上传/下载进度与历史记录。 | ✅ |
| 传输中心 | 支持快速打开传输项对应的本地文件位置。 | ✅ |
| 触发器 | 支持建立多套规则，基于关键字匹配后可自动输入并发送配置内容。 | ✅ |
| 触发器 | 高级匹配规则与更广泛自动化场景仍在持续完善。 | ⚠️ |
| i18n | 终端、文件、设置与 AI 核心流程支持中英文，覆盖范围仍在扩展。 | ⚠️ |
| 平台 | 原生平台体验与深度系统集成能力持续打磨中。 | ⚠️ |
| 平台 | 全平台验证仍在推进（当前优先验证 macOS，Windows/Linux 待补齐）。 | ⚠️ |

状态说明：`✅ 已可用` · `⚠️ 进行中 / 部分可用` · `❌ 规划中`
